# Portfolio Website

This is a minimal, modern portfolio website scaffold. Replace the placeholder details in `script.js` with your real information and add your resume to `resume.pdf`.

Quick start (open locally):

1. Open [index.html](index.html) in a browser.
2. Or serve locally (Python):

```bash
python -m http.server 8000
# then open http://localhost:8000
```

What to edit:
- `script.js` — set `PROFILE` fields (name, links, resume, photo) and update `PROJECTS`.
- `images/profile.svg` — replace with your image (or change `PROFILE.photo` path).
- `resume.pdf` — replace with your resume PDF to enable the download button.
